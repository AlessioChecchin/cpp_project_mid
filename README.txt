Intermediate exam project of TonioloChecchin group
Members: Alessio Checchin, Jacopo Toniolo, Giulio Cattelan

The project is made of a Book class, a Date class and a Boookshelf class.

The Book class meets all the requirements of assignment.
The Date class handles dates from 1/1/1582.
The Boookshelf class is a copy of std::vector but it can only handle Book UDTs.
